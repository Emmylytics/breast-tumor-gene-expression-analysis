#### Breast Tumor Gene Expression Analysis

This project analyzes gene expression profiles in breast tumor and normal tissues to identify key genetic differences associated with cancer development. The analysis includes data preprocessing, statistical testing, and visualization of differential expression patterns.

---

#### Objective & Key Findings

This analysis explores variations in gene expression between tumor and normal tissue samples to identify potential biomarkers and underlying biological differences.
- While overall expression patterns were broadly consistent, certain genes displayed marked deviations between tumor and healthy tissues.
- These differentially expressed genes present potential candidates for early tumor detection and progression tracking.
- Statistical evaluation using (t-tests) helped isolate genes with significant changes in expression levels.
- Correlation analysis and barplot visualizations provided deeper insight into gene interactions and intensity trends across this dataset.
